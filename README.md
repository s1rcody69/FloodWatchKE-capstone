#  FloodWatchKE

**FloodWatchKE** is a community-driven flood reporting and awareness web application designed to help residents of Nairobi report flooding incidents in real time and keep others informed about dangerous areas.

This project was developed as a ** Project using HTML, CSS, and JavaScript.**

---

##  Problem

Flooding is a frequent issue in many parts of Nairobi, especially in areas near rivers and informal settlements. One major challenge is the **lack of real-time, localized information** about flood conditions.

People often rely on:
- Word of mouth
- Social media posts
- Delayed news reports

This delay can put communities at risk.

---

##  Solution

FloodWatchKE provides a **simple platform where community members can:**
- Report flood incidents
- View reports from other users
- Confirm existing reports
- Understand severity levels quickly

This helps people **make safer decisions** during heavy rains and flooding.

---

##  Key Features

- Community flood reporting
- Real-time report display
- Severity classification system
- Confirmation system for reports
- Clean and modern user interface
- Mobile-friendly design

---

##  Severity Levels

FloodWatchKE uses a color-coded system to indicate danger levels:

| Level | Meaning |
|------|--------|
| 🟡 Watch | Low risk – Monitor situation |
| 🟠 Warning | Moderate risk – Be cautious |
| 🔴 Danger | High risk – Immediate action needed |
